### [1] Double comparison

```cpp
const double EPS = 1e-9;
int comp_double(double a, double b){
  if(fabs(a - b) <= EPS){
     return 0; // equal
  }
  return (a < b) ? -1 : 1; 
}
```

### [2] Get ceil for `a/b`
> Instead of using `ceil` builtin function in cpp use the following formula.
```cpp
ceil(a / b) = (a + b - 1) / b   //integer domain instead of double domain
```
